********************
get-game-session-log
********************



===========
Description
===========

Download a compressed log file for a game session.



========
Synopsis
========

::

    get-game-session-log
  --game-session-id <value>
  --save-as <value>




=======
Options
=======

``--game-session-id`` (string)
The game session ID

``--save-as`` (string)
The filename to which the file should be saved (.zip)
