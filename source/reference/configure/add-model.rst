*********
add-model
*********



===========
Description
===========

describe the command



========
Synopsis
========

::

    add-model
  --service-model <value>
  [--service-name <value>]




=======
Options
=======

``--service-model`` (string)
The contents of the service JSON model.

``--service-name`` (string)
Overrides the default name used by the service JSON model to generate CLI service commands and Boto3 clients.
