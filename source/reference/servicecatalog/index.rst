**************
servicecatalog
**************



===========
Description
===========

 

 **Overview**  

 

 `AWS Service Catalog <https://aws.amazon.com/servicecatalog/>`_ allows organizations to create and manage catalogs of IT services that are approved for use on AWS. This documentation provides reference material for the AWS Service Catalog end user API. To get the most out of this documentation, be familiar with the terminology discussed in `AWS Service Catalog Concepts <http://docs.aws.amazon.com/servicecatalog/latest/adminguide/what-is_concepts.html>`_ .

 

 *Additional Resources*  

 

 
* `AWS Service Catalog Administrator Guide <http://docs.aws.amazon.com/servicecatalog/latest/adminguide/introduction.html>`_   
 
* `AWS Service Catalog User Guide <http://docs.aws.amazon.com/servicecatalog/latest/userguide/introduction.html>`_
