acm

===========
Description
===========

 

Welcome to the AWS Certificate Manager (ACM) CLI Command Reference. This guide provides descriptions, syntax, and usage examples for each ACM CLI command. You can use AWS Certificate Manager to request ACM Certificates for your AWS-based websites and applications. For general information about using ACM and for more information about using the console, see the `AWS Certificate Manager User Guide`_ . For more information about using the ACM API, see the `AWS Certificate Manager API Reference`_ .
